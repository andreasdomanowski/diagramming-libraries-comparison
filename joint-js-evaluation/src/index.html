JointJS Evaluation

Requirements for the editor can be found here. Inside the container canvas, the context menu can be used to create new shapes. A double click on an element selects it to be a link source. A click on another element then selects this element as the link target and draws the link. Labels can be edited by a context menu click on an element.

The purpose of this demo is just to get in touch with the API of JointJS - and to refresh my rusty knowledge of JS in general as well as to get used to Webpack and the Node ecosystem.
